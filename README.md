# SpaceShooter

Prueba Técnica Unity Developer - Proyecto "Space Shooter"

Editor de Unity versión 2022.3.62f2

Editor de código Visual Studio 2022



Controles:

* Moverse - WASD
* Disparar - Espacio
* Pausar - P



Este proyecto utiliza los siguientes paquetes externos instalados manualmente:



Fuente - Unity Registry:

* Input System 1.14.2



Fuente - Unity Asset Store:

* Space Shooter FREE 1.0 (EXCLUYENDO ARCHIVOS DE CONFIGURACIÓN, DOCUMENTACIÓN, ESCENAS, SCRIPTS Y PREFABS)
* DOTween (HOTween v2) 1.2.825
* Game Music Stingers and UI SFX 2 Pack 1.0
